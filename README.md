# Ashtakoot Milan

This project contains a single HTML page for Ashtakoot matching.

## Project File

- `Ashtakoot_Milan_Fixed.html`

## How to Use

1. Open `Ashtakoot_Milan_Fixed.html` in any modern web browser.
2. Enter the required details in the form.
3. View the generated matching output on the page.

## Notes

- This is a static HTML file and does not require any installation.
